# neumann-docs

経営管理プロダクト領域におけるAgentAI

> **Note**: コードリポジトリは [neumann](../neumann/) を参照してください。

## 命名由来

ジョン・フォン・ノイマン（John von Neumann, 1903-1957）

- **ゲーム理論の創始者**: 経営戦略・意思決定の数学的基盤を確立
- **計算機科学の父**: 現代コンピュータアーキテクチャ（ノイマン型）を設計
- **経済学への貢献**: 期待効用理論により合理的意思決定を定式化

経営管理とAIの交差点に立つプロジェクトとして、両領域の先駆者の名を冠する。

## プロジェクト構成

```
neumann/
├── 00_vision/        # ビジョン・思想（Human主導）
├── 01_concept/       # コンセプト定義
├── 02_product/       # プロダクト設計
├── 03_validation/    # 仮説検証
├── 04_business/      # 事業計画
├── 05_decisions/     # 意思決定記録
└── _workspace/       # AI作業領域
```

## 協働モデル

| 領域 | Human | AI |
|------|-------|-----|
| Vision | 思想・方向性の決定 | 言語化支援 |
| Concept | 仮説の承認 | リサーチ・構造化 |
| Product | 優先度判断 | 設計・詳細化 |
| Validation | 実験承認・解釈 | 仮説整理・分析 |
| Business | 戦略的意思決定 | 数値モデル・調査 |
| Decisions | 最終承認 | 記録・追跡 |
