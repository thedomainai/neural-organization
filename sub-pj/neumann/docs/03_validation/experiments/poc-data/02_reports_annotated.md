# Weekly Reports - Annotated for Ambiguity Detection

各報告の曖昧性パターンをアノテーション。検出精度測定の正解データとして使用。

---

## アノテーション凡例

| Pattern | Code | Description |
|---------|------|-------------|
| 浅い分析 | `[SA]` | Shallow Analysis - 原子レベルまで分解されていない |
| カバレッジ不足 | `[MC]` | Missing Coverage - 必要な項目が報告されていない |
| 定量化不足 | `[LQ]` | Lack of Quantification - 数値がない、定性的 |
| アクション不明確 | `[UA]` | Unclear Action - Who/What/When が不明 |
| 事実と解釈の混同 | `[FI]` | Fact/Interpretation Mixing - 主観と客観が分離されていない |

**重要度**:
- `Critical`: 意思決定に直接影響する曖昧性
- `Warning`: 追加情報が望ましい曖昧性
- `Info`: 軽微だが改善の余地がある曖昧性
