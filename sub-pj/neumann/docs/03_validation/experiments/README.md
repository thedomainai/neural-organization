# Experiments

## 構成

```
experiments/
├── README.md           # このファイル
├── exp-001-xxx.md      # 実験記録
└── ...
```

## 実験テンプレート

各実験は以下の形式で記述:

```markdown
# EXP-XXX: 実験名

## 検証する仮説
- 仮説ID:
- 仮説内容:

## 実験設計
- 方法:
- サンプルサイズ:
- 成功基準:
- 期間:

## 実行記録
- 開始日:
- 完了日:
- 実施内容:

## 結果
- データ:
- 分析:

## 結論
- 仮説の判定: 検証/棄却/継続検証
- 学び:
- 次のアクション:
```
