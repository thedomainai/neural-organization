# 人事制度：コンピテンシー・ディクショナリー Ver 3.0

## 1. 制度概要

### 1.1 コンピテンシー体系
本制度では、成果を生み出すための普遍的な行動ドライバーを以下の3領域・6要素で定義する。

| 領域 | コンピテンシー | 定義 |
| :--- | :--- | :--- |
| **1. 思考・構想**<br>(Thinking) | **A. 課題設定**<br>(Issue Finding) | 状況を観察し、「何を解決すべきか（問い）」を定義する力 |
| | **B. 構造化・計画**<br>(Structuring) | 複雑な事象を整理し、解決までの論理的な道筋（解き方）を描く力 |
| **2. 実行・行動**<br>(Action) | **C. スピード**<br>(Speed) | 意思決定から実行、完了までのサイクルを回す速さとリズム |
| | **D. 品質**<br>(Quality) | アウトプットの正確性、完成度、および持続可能性（強固さ） |
| **3. 影響・対人**<br>(Relation) | **E. 受信・傾聴**<br>(Listening) | 他者の発言、感情、および組織の文脈を深く正確に読み取る力 |
| | **F. 発信・伝達**<br>(Delivery) | 自身の意図やビジョンを相手に届け、納得と行動を引き出す力 |

### 1.2 グレードとスコープ（責任範囲）
等級（グレード）は役職任命とは独立しており、以下の責任範囲（スコープ）においてコンピテンシーが発揮されているかで判定される。

| グレード | 役割名称 | 責任・影響のスコープ (Scope of Impact) |
| :--- | :--- | :--- |
| **G8** | **取締役級** | **【業界・社会 (Industry / Society)】**<br>国内市場や産業構造全体に変革をもたらし、企業の社会的プレゼンスを確立する。 |
| **G7** | **執行役員級** | **【全社・経営 (Whole Company / Management)】**<br>全社の経営資源、組織構造、企業文化、および中長期の事業ポートフォリオに責任を負う。 |
| **G6** | **本部長級** | **【本部・事業部 (Division / Business Unit)】**<br>複数の「部」を束ねる事業単位、または特定の事業ドメイン全体の戦略と成果、組織力に責任を負う。 |
| **G5** | **部長級**<br>(Manager) | **【部・機能 (Department / Function)】**<br>特定の部門や専門機能全体の最適化、および半年〜1年スパンのKPI達成に責任を負う。 |
| **G4** | **マネージャー<br>サポート** | **【プロジェクト群・チーム間連携 (Project / Cross-Team)】**<br>複数のチームやプロジェクトを横断的に支援し、部長の右腕として現場の品質と進行を統括する。 |
| **G3** | **リーダー** | **【チームリード (Team Leadership)】**<br>特定のチーム（数名〜）を牽引し、メンバーの成果、品質、およびデリバリー速度に責任を負う。 |
| **G2** | **リーダー<br>サポート** | **【自律・技術牽引 (Autonomous / Tech Lead)】**<br>自身の高い専門性でチームの難題を解決し、実務面でリーダーを支え、周囲の手本となる。 |
| **G1** | **メンバー** | **【担当タスク・定型業務 (Task / Duties)】**<br>指示された業務を完遂し、プロとしての基本動作を身につけ、チームに貢献する。 |

## 2. グレード別コンピテンシー定義詳細

### 2.1 思考・構想 (Thinking)

| グレード | A. 課題設定 (Issue Finding) | B. 構造化・計画 (Structuring) |
| :--- | :--- | :--- |
| **G8** | **【産業構造の再定義】**<br>自社のみならず、業界全体の商習慣やルール、サプライチェーンの構造的な歪みを課題として捉える。既存の産業定義を書き換え、新たなゲームチェンジを仕掛ける「問い」を立てる。 | **【エコシステム構想】**<br>一企業では完結しない壮大なビジョンを描き、競合他社、行政、異業種をも巻き込んだエコシステム全体を設計する。業界標準となるプラットフォームやアライアンスを構築する。 |
| **G7** | **【事業創造と破壊】**<br>全社視点で既存事業の限界やリスクを見抜き、自らの手で事業をピボット（方向転換）させる決断をする。サンクコストに囚われず、撤退や統合の判断も行う。 | **【全社資源の最適配分】**<br>ヒト・モノ・カネ・情報の経営資源を、部門の壁を超えてダイナミックに再配分する戦略を描く。短期的な痛みと引き換えに、中長期的な競争優位性を確立するためのロードマップを策定する。 |
| **G6** | **【構造的課題の特定】**<br>本部内や事業部が抱える構造的な矛盾（組織のサイロ化、技術的負債の蓄積など）の根本原因を特定する。仕組みや構造そのものを変えることで解決すべきイシューを定義する。 | **【事業ロードマップ策定】**<br>単年度のPL責任を超え、3〜5年スパンでの事業や組織能力の進化プロセスを設計する。不確実な未来に対し、複数のオプションを持たせた柔軟かつ強靭な計画を立案する。 |
| **G5**<br>*(部長)* | **【部門方針の策定】**<br>全社戦略と連動させつつ、自部門が半年〜1年で達成すべきゴールとKPIを設定する。メンバーが迷わず走れるよう、抽象的な戦略を具体的な部門目標へ落とし込む。 | **【リソース戦略】**<br>部門目標の達成に必要な採用計画、予算配分、体制作りを設計する。突発的なリスクや環境変化を見越して、リソースにバッファ（ゆとり）や代替案を持たせた計画を組む。 |
| **G4**<br>*(Mgrサポ)* | **【プロジェクト群の最適化】**<br>複数のチームやプロジェクトにまたがる共通課題（重複業務、連携ミスなど）を発見し、横断的な解決策を提案する。部長の視点を補完し、現場のリアリティを踏まえた課題設定を行う。 | **【複雑な工程の交通整理】**<br>依存関係が複雑な複数プロジェクトのスケジュールを統合的に管理する。クリティカルパスを特定し、チーム間の干渉や手戻りを防ぐための全体工程表を描く。 |
| **G3**<br>*(リーダー)* | **【チーム課題の発見】**<br>チームの目標達成を阻害している要因（スキル不足、ツール不備、士気低下など）を特定する。上司から言われる前に、現場レベルでの改善テーマを自ら設定する。 | **【実行プランの具体化】**<br>チームの目標を個人のタスクに分解し、誰が・いつまでに・何をやるかを明確にする。メンバーの能力を考慮した現実的なスケジュールと役割分担を組む。 |
| **G2**<br>*(Ldrサポ)* | **【自律的な解決策提示】**<br>自身の担当領域において、困難な壁にぶつかっても、自分で調べ、考え、複数の解決案（松竹梅）を持って上司に相談できる。「どうすればいいですか？」ではなく「こうしたい」を提案する。 | **【ロジカルな段取り】**<br>複雑なタスクであっても、ロジックに基づいて手順を組み立て、効率的に進める段取りができる。自分の作業の依存関係を理解し、他者を待たせない工夫をする。 |
| **G1**<br>*(メンバー)* | **【違和感のアラート】**<br>業務遂行中に発生した小さな違和感や異常を放置せず、速やかに報告・相談できる。 | **【手順の理解と遵守】**<br>指示された業務の背景・目的・手順を正しく理解し、マニュアルやルールに沿って忠実に計画をなぞることができる。 |

### 2.2 実行・行動 (Action)

| グレード | C. スピード (Speed) | D. 品質 (Quality) |
| :--- | :--- | :--- |
| **G8** | **【市場モメンタムの創出】**<br>自社のリリースが業界全体のトレンドやタイムラインを決定づける。「当社が動けば業界が動く」というレベルで、市場投入のタイミングを戦略的に操る。 | **【デファクトスタンダード】**<br>当社のプロダクトや技術、手法が、事実上の業界標準となる品質を作り上げる。「この領域といえば〇〇」と社会的に認知される最高峰のブランドを確立する。 |
| **G7** | **【組織OSの再設計】**<br>全社の意思決定プロセスそのものを再設計し、大企業病を防ぐ。リスクを取って即断即決するカルチャーを浸透させ、市場変化のスピードを凌駕する組織リズムを作る。 | **【変革の完遂とガバナンス】**<br>痛みを伴う改革や難易度の高いプロジェクトを完遂させる。同時に、企業の社会的信用を守るためのガバナンスを効かせ、スピードとコンプライアンスを両立させる。 |
| **G6** | **【デリバリー速度の最大化】**<br>本部全体の生産性を阻害するプロセスや慣習を撤廃し、アウトプットまでのリードタイムを劇的に短縮する。自動化、権限委譲を推進し、組織としての機動力を高める。 | **【品質基準の確立】**<br>「当社における品質とは何か」という高い基準を定義し、妥協を許さない文化を作る。管掌範囲の専門性を統括し、業界トップレベルの技術的・実務的品質を保証する。 |
| **G5**<br>*(部長)* | **【意思決定の迅速化】**<br>部内の決裁や承認を停滞させず、メンバーを待たせない。不完全な情報下でも、経験と勘所に基づいて「Go/No-Go」の判断を遅滞なく下す。 | **【成果責任の担保】**<br>部門としてのアウトプット品質に最終責任を持つ。トラブル時には自ら前線に立ち、火消しと事態収拾を完遂し、顧客や関連部署からの信頼を守り抜く。 |
| **G4**<br>*(Mgrサポ)* | **【連携スピードの向上】**<br>チーム間のボールの受け渡し（連携）をスムーズにし、ポテンヒットや待ち時間をゼロにする。横断的な調整役として立ち回り、プロジェクト全体の進行を加速させる。 | **【クロスチェックと標準化】**<br>チームを跨いで品質のバラつきを是正する。ベストプラクティスを横展開し、特定のチームだけでなく、関わる全プロジェクトの品質底上げに貢献する。 |
| **G3**<br>*(リーダー)* | **【チーム作業の効率化】**<br>メンバーの手戻りや迷いを減らし、チーム全体のアウトプット速度を引き上げる。会議の効率化やツールの導入など、足元の生産性向上を主導する。 | **【レビュアー責任】**<br>メンバーの成果物に対して責任を持ち、リリース基準をクリアさせる。レビューを通じてメンバーの品質意識を高め、ミスを未然に防ぐゲートキーパーとなる。 |
| **G2**<br>*(Ldrサポ)* | **【安定した自走】**<br>リーダーの細かい指示がなくとも、一定のスピードで自走できる。突発的なトラブルや負荷がかかっても、パニックにならず冷静に対処し、納期を守る。 | **【高い専門性と自己完結】**<br>自身の得意領域においてはリーダーをも凌ぐ専門性を発揮し、高品質なアウトプットを独力で出す。「技術的なことは〇〇さんに聞けば安心」という信頼を得る。 |
| **G1**<br>*(メンバー)* | **【迅速な着手】**<br>指示を受けたら「まず着手する」習慣がある。完璧主義に陥って止まることなく、プロトタイプやドラフトを早く出して方向性を確認する姿勢を持つ。 | **【基本動作の徹底】**<br>確認、検証、自己レビューといった基本動作を怠らない。不確かなまま進めず、必ず確認をとってから実行する慎重さを持つ。 |

### 2.3 影響・対人 (Relation)

| グレード | E. 受信・傾聴 (Listening) | F. 発信・伝達 (Delivery) |
| :--- | :--- | :--- |
| **G8** | **【時代精神の感応】**<br>市場の声だけでなく、社会倫理、次世代の価値観、地政学的な変化など、より大きな文脈を深く読み解く。社会が企業に求める「公器」としての役割を直感的に捉える。 | **【ソートリーダーシップ】**<br>業界の第一人者として発信し、業界全体の技術トレンドや思想をリードする。その発言が市場や行政を動かし、新たな規制や標準を作る力を持つ。 |
| **G7** | **【組織力学の洞察】**<br>組織内に渦巻く政治的力学、隠れた対立構造、社員の集合的無意識を読み解く。表面的な報告に惑わされず、組織の「真実」を掴む洞察力がある。 | **【ナラティブによる牽引】**<br>数字や論理だけでなく、企業の歴史や使命（ミッション）に基づいた「物語」を語り、組織全体を熱狂させ、大きな方向性へと動かす求心力を発揮する。 |
| **G6** | **【メタ認知と他者視点】**<br>自分や自本部が、全社（他本部・経営陣）からどう見られているかを客観的に認識できる。独善的な判断を避け、全体最適の視点から自組織の立ち位置を修正できる。 | **【概念の言語化】**<br>暗黙知となっている成功法則や、曖昧な組織文化を、明確な言葉（スローガン、行動指針）に落とし込む。その言葉を繰り返し発信し、組織の共通言語として定着させる。 |
| **G5**<br>*(部長)* | **【心理的安全性と本音】**<br>部下の不安や不満、キャリアへの悩みを引き出し、安心して挑戦できる環境を作る。1on1等で信頼関係を築き、離職の兆候や組織の歪みを早期に察知する。 | **【合意形成と交渉】**<br>他部署との利害対立やリソースの奪い合いにおいて、粘り強く交渉し、自部門にとって有利かつ全社的にも納得感のある落とし所（Win-Win）をまとめる。 |
| **G4**<br>*(Mgrサポ)* | **【ハブとしての傾聴】**<br>リーダー層からの突き上げと、部長層からのトップダウンの両方の意見を聞き、翻訳して繋ぐ。現場のリアリティを経営層に正しく伝えるパイプ役となる。 | **【フォロワーシップと伝播】**<br>部長の方針を咀嚼し、現場のリーダーたちが理解しやすい言葉に変換して伝える。批判的なメンバーに対しても粘り強く説明し、組織の方針に従わせる影響力を持つ。 |
| **G3**<br>*(リーダー)* | **【メンバーのコンディション把握】**<br>日々の会話からメンバーの体調やモチベーションの変化に気づく。業務上の悩みだけでなく、人間関係や環境面での障壁を取り除くために耳を傾ける。 | **【動機付けとフィードバック】**<br>メンバー一人ひとりの性格や志向に合わせて、やる気を引き出す声がけをする。ポジティブな評価だけでなく、耳の痛い改善点も、相手の成長のために率直に伝える。 |
| **G2**<br>*(Ldrサポ)* | **【背景理解】**<br>指示や依頼を受けた際、単なる作業内容だけでなく、その背景にある「目的」「重要度」を正しく理解する。リーダーの意図を汲み取り、先回りして動く。 | **【論理的な報告】**<br>事実（Fact）と解釈（Opinion）を明確に分けて報告できる。トラブル時にも感情的にならず、状況を冷静かつ正確に伝え、上司の判断を仰ぐことができる。 |
| **G1**<br>*(メンバー)* | **【傾聴の姿勢】**<br>相手の話を遮らず、最後まで目を見て聞くことができる。メモを取る、相槌を打つなど、相手に対するリスペクトを行動で示せる。 | **【明確な意思表示】**<br>挨拶、返事、感謝、謝罪などの基本的なコミュニケーションをハッキリと行える。「わかりました」「できません」の意思表示を曖昧にしない。 |
