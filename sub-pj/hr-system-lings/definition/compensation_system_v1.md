# 人事制度：報酬制度 (Compensation System)

## 1. 報酬ポリシー
本制度は、透明性と納得感を最大化するため、以下のポリシーで運用する。

*   **完全等級連動型**: 年収は「グレード（能力）」と「役職（責任）」によって機械的に決定される。属人的な調整（入社経緯や前職給与など）による不公平を排除する。
*   **年俸制（Annual Salary）**: 賞与（ボーナス）変動による生活の不安定さを排除し、プロフェッショナルとして約束された年収を12分割で毎月支給する。
*   **洗い替え型**: 毎回の評価によりステップ（S1-S5）が変動する。過去の実績ではなく「今の貢献度」で年収が決まる。

## 2. 報酬構成
**「基本年俸（Base Salary）」＋「役職手当（Role Allowance）」** の総額管理。
通勤交通費（実費）以外の住宅手当・家族手当等は設けない。

## 4. 役職手当（月額・年額）

## 6. 運用ルール（昇給・降給）
評価結果に基づき、以下のルールでステップを移行する。

*   **昇格時**: 上位グレードのS1（または現年収を上回る直近のステップ）へ移行。
*   **評価による号棒改定（年1回想定）**:
    *   **Outstanding (5)**: 2ステップアップ
    *   **Exceed (4)**: 1ステップアップ
    *   **Meet (3)**: ステイ（維持）
    *   **Below (2)以下**: 1ステップダウン（または降格検討）
